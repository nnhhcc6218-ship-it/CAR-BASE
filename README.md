# 🚗 CAR BASE - 브릴라 코리아 팀명지

**차량 외장관리 전문샵 - CAR BASE의 공식 웹사이트**

## 📋 소개

CAR BASE는 차량의 가치를 높이는 프리미엄 차량 외장관리 전문 서비스를 제공하는 브릴라 코리아 팀명지의 공식 브랜드입니다.

### 서비스 분야
- ✨ **디테일링 세차**: 전문가의 손길로 하는 세밀한 세차
- ☀️ **광택 시공**: 도장면 복구 및 완벽한 광택
- 🛡️ **유리막 코팅**: 나노 유리막 코팅으로 장시간 보호
- 💨 **실내 크리닝**: 시트, 대시보드, 에어컨 청소
- 🧹 **특수 크리닝**: 산화, 타르 등 특수 오염 제거
- 📦 **에바 크리닝**: 에바 매트 정밀 세정
- 🎓 **교육**: 자동차 관리 전문가 양성 프로그램
- 💧 **세차용품 판매**: 프리미엄 케미컬 제품

---

## 🌐 웹사이트 기능

### 주요 섹션
1. **네비게이션**: 빠른 이동 메뉴
2. **홈(Hero)**: 임팩트 있는 첫 인상
3. **서비스**: 8가지 핵심 서비스 소개
4. **포트폴리오**: 시공 사례 갤러리
5. **소개**: 브랜드 정보 및 연락처
6. **소셜 미디어**: 모든 채널 통합
7. **상담 신청**: 예약 및 문의 폼
8. **푸터**: 빠른 링크 및 정보

### 기술 특성
- 📱 **반응형 디자인**: 모든 기기에서 최적화
- ⚡ **빠른 로딩**: 경량 최적화
- 🎨 **모던 디자인**: 프리미엄 브랜드 이미지
- 🔍 **SEO 최적화**: 검색 엔진 친화적
- ♿ **접근성**: WCAG 표준 준수
- 🎯 **사용자 경험**: 직관적인 네비게이션

## 🚀 시작하기

### 로컬 실행
1. 저장소를 클론합니다:
```bash
git clone https://github.com/nnhhcc6218-ship-it/CAR-BASE.git
cd CAR-BASE
```

2. 로컬 서버로 실행합니다:
```bash
# Python 3 사용
python -m http.server 8000

# 또는 Python 2 사용
python -m SimpleHTTPServer 8000
```

3. 브라우저에서 열기:
```
http://localhost:8000
```

### GitHub Pages로 배포
1. Settings > Pages로 이동
2. Source를 "main" 브랜치로 설정
3. 저장하면 자동으로 배포됨

## 🎨 커스터마이징 가이드

### 브랜드 색상 변경
`styles.css`의 `:root` 섹션을 수정하세요:

```css
:root {
    --primary-color: #1a1a2e;      /* 주색상 */
    --secondary-color: #e94560;    /* 보조색상 */
    --accent-color: #16c784;       /* 강조색상 */
}
```

### 콘텐츠 업데이트
- `index.html`의 각 섹션을 직접 편집하세요
- 이미지는 추가하려는 경우 `/images` 폴더를 만들어 사용하세요

### 연락처 폼 설정
`script.js`에서 `contactForm` 처리 부분을 수정하여 실제 이메일 서비스와 연동할 수 있습니다.
